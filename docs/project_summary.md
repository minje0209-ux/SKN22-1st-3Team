# 국내 자동차 시장 트렌드 분석

## 1. 프로젝트 목적

현대 및 기아 자동차 모델을 중심으로 **시장 관심도(Interest)**와 **실제 보급률(Adoption)**을 비교 분석하는 프로젝트. 다양한 웹/공공 데이터를 결합하여 **모델별 트렌드 변화, 시장 반응, 실제 보급 구조**를 시각적으로 이해하는 것을 목표로 한다.

---

## 2. 분석 지표 정의

### 2-1. 관심도(Interest Index)

사람들이 특정 모델에 대해 어느 정도 관심을 보였는지를 정량화한 지표.

**데이터 출처**

* 네이버 데이터랩 검색량
* 구글 트렌드 비교 지수
* 다나와 모델 조회수

**특징 및 주의점**

* 구글 트렌드는 절대 검색량이 아닌 상대 지수이므로 가중치 조정 필요.
* 네이버 데이터랩은 국내 비중이 높아 중요 기준으로 사용.
* 플랫폼별 노이즈가 다르므로 정규화 작업 필요.

**예시 산식**

```
관심도 =
0.5 × 네이버 검색량 비중
+ 0.4 × 구글 트렌드 지수
+ 0.1 × (다나와 인기순 rank를 점수로 변환한 값)
```

```
# 예: 한 달에 같은 세그먼트 내에서 최대 50위까지 있다고 가정
다나와 인기 점수 = 1 - ( (rank - 1) / (max_rank - 1) )

rank = 1  → 1.0
rank = 25 → 0.5
rank = 50 → 0.0
```

---

### 2-2. 보급률(Adoption Rate)

해당 모델이 실제 시장에서 얼마나 판매되었는지를 나타내는 지표.

**데이터 출처**

* 다나와 자동차 판매실적(월별 출고량)
* 공공데이터 자동차 신규등록 통계(보조적 활용)

**설명**

* 국내에는 현대/기아의 정확한 모델별 판매량 API가 없음.
* 다나와 판매실적이 가장 신뢰도 있는 “민간 기반 판매량 근사치”.
* 공공데이터의 등록현황은 *판매량이 아니라 등록 완료된 차량 숫자*이므로 직접 매핑은 어려우나, **시장 흐름 측정에는 유효**.

**보조 활용 아이디어**

* 연료별·차종별 등록 변화를 통해 전기차/하이브리드 트렌드 파악
* 특정 분기·지역별 등록 변화로 시장 전반의 흐름 보정
* 신규등록 증감률을 보급률 가중치로 활용 가능

**산식**

```
보급률 = (모델 월 판매량 / 전체 월 판매량) × 100
```

## 3. 공공데이터 활용 가이드(선택)

보급률 산식에 직접 반영하기는 어렵지만, 아래 목적에서는 충분히 의미 있는 데이터로 활용 가능하다.

### 3-1. 시장 구조 및 흐름 분석

* 연료별 등록 추이 → EV, HEV, LPG 시장 변동 파악
* 세그먼트별 등록 수 → SUV, 세단 시장의 규모 변화
* 지역별 등록 분포 → 특정 지역에서 EV·HEV 성장률 확인

### 3-2. 관심도 및 보급률 보정

* 다나와 판매실적이 특정 모델 중심으로 쏠려 있거나 노이즈가 있는 경우, 공공데이터의 **연간/월별 등록 성장률**을 기준으로 전체 시장 성장/축소를 보정할 수 있음.

### 3-3. 제조사 전략 분석(간접적 활용)

* 친환경차 등록 증가 → 현대/기아의 EV 라인 확대 흐름 분석
* 경량 차종 감소 → SUV 중심 전략 강화 추정 등

## 7. 결론 요약

* 관심도는 검색량·조회수 기반 지수.
* 보급률은 다나와 판매실적 기반 시장 점유율.
* 공공데이터는 직접적 보급률 계산에는 적합하지 않으나, **시장 전체 흐름 설명·보정·확장 분석**에 매우 유용.
* 다나와, 네이버 데이터랩, 구글 트렌드를 통합하여 모델별 트렌드 분석 수행.
