# 🚗 국내 자동차 시장 트렌드 분석

### 현대자동차/기아자동차 - 관심도 vs 보급률 프로젝트 요약

------------------------------------------------------------------------

## 1. 프로젝트 목적

현대·기아 차량을 중심으로 **시장 관심도(Interest)**와 **실제
보급률(Adoption)**의 차이를 분석하는 데이터 분석 프로젝트.
웹/공공 데이터를 통합해 **모델별 트렌드 변화**를 시각화하는 것이 목표.

------------------------------------------------------------------------

## 2. 분석 지표 정의

### ✅ 관심도(Interest Index)

> "사람들이 특정 모델에 대해 얼마나 관심을 보였는가"

**데이터 출처**
- 네이버 데이터랩 검색량
- 다나와 모델 조회수
- 현대/기아 공식 API 기반 페이지 호출 추정치(선택)
- Google Trends 비교 지수



**관심도 형식 예시**

    관심도 =
    0.5 × 네이버 검색량 비중
    + 0.4 × 구글 검색 트렌드 지수
    + 0.1 × 다나와 인기순위

------------------------------------------------------------------------

### ✅ 보급률(Adoption / Penetration Rate)

> "해당 모델이 실제 시장에서 얼마나 판매되었는가"

보급률은 **다나와 자동차 판매실적(월별 출고량)**을 기준으로 계산.

**데이터 출처** - 다나와 판매실적:
https://auto.danawa.com/newcar/?Work=record
- (옵션) 공공데이터 신규등록(비중 조절 가능)

**보급률 공식**

    보급률 = (모델 월 판매량 / 전체 월 판매량) × 100

------------------------------------------------------------------------

## 3. 프로젝트 범위(Boundary)

**포함**
 - 현대/기아 승용차 (세단/SUV/전기차)
 - 가격, 연비, 연료 데이터 포함
 - 최신 연식 기준

**제외**
- 트림 단위 분석 제외
- 포터/봉고 등 상용차 제외
- 중고차, 개인 시세 제외

-   **관심도 = 검색량/조회수 기반 지수**
